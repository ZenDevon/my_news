## 前言

这是一个使用原生的JavaScript编写的新闻头条多页面应用，使用组件化进行开发。

## 技术栈

nodejs + express + mysql + webpack5 + es6

## 项目运行

```bash
git clone https://github.com/ZenDevon/my_news.git

# 启动服务器
cd news_server
npm i
npm run start

# 启动项目(本地预览)
npm i
npm run dev
```
